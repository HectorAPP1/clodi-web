# Landing Clodi

Página de landing para Clodi

## Descripción

Proyecto de landing page desarrollado con HTML, CSS y JavaScript.

## Estructura del Proyecto

```
Landing Clodi/
├── index.html          # Página principal
├── assets/            # Recursos estáticos
│   ├── clodi_ISOTIPO.svg
│   └── clodilogo.svg
└── README.md          # Este archivo
```

## Características

- Diseño responsive
- Logotipos SVG optimizados
- Estructura HTML semántica

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/HectorAPP1/Landing-Clodi.git
```

2. Abre `index.html` en tu navegador

## Uso

Simplemente abre el archivo `index.html` en cualquier navegador web moderno.
